# 在线与离线模拟考试站

基于本地 HTML 试卷整理的静态学习网站，可通过 GitHub Pages 在线访问，也可安装后离线使用。

- 共 11 套试卷、665 道题
- 5 套大数据试卷和 3 套软件工程试卷、共 519 道题支持快速刷题与参考答案
- 支持纵向快速刷题和每组 20 题的闯关模式
- 主观题通过“会做 / 不会”自评，客观题支持同义答案判定
- 支持 PWA 安装、离线缓存和版本更新提示
- 答题记录保存在当前浏览器的本地存储中
- 不需要服务器端程序或数据库

在线地址：

https://bluesky2637.github.io/big-data-exam/

## 说明

本项目仅用于个人学习与复习。试卷专业方向和参考答案为根据题目内容及教材整理，不代表学校官方认定或官方答案。

所有页面均设置 `noindex,nofollow`，用于降低被搜索引擎收录的概率，但不等同于访问密码保护。

## 验证

```powershell
npm test
```

自动检查题数、答案覆盖、选项与填空结构、同义答案、主观题、内部链接、PWA 资源和脚本语法。
